# Galois Cohomology Computation Engine

A Python library for computing group cohomology H^n(G, M), with emphasis on Galois cohomology of number fields.

## Features (planned)

- Cochain complexes and coboundary operators for finite groups
- Bar resolutions and projective resolutions
- Lyndon-Hochschild-Serre spectral sequence
- Long exact sequences with connecting homomorphisms
- Applications: Hilbert 90, Brauer groups, Galois descent, Tate cohomology

## Setup

```bash
pip install -e ".[dev]"
```

## Testing

```bash
pytest
```
